[![CircleCI](https://circleci.com/gh/reindeer-project/reindeer-token.svg?style=svg)](https://circleci.com/gh/reindeer-project/reindeer-token)

reindeer token

# Requirements

node.js, yarn

# Installation

`git clone https://github.com/reindeer-project/reindeer-token.git1`

`cd reindeer-token`

`yarn`

`yarn install`

# Testing

`yarn lint:sol`

`yarn test:cov`
